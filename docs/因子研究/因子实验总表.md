### 本轮实证复审

本轮用 2024-01-01 至 2026-05-15 的 workspace 日频 bars 对业绩预告策略做角色循环和 overlay 复核：

| 研究线 | 结果 | 结论动作 |
|---|---|---|
| 坏持仓过滤 `bad_holder` | 5 个候选全部 rejected；`range_vol_20d` 有 10d RankIC 0.0638 但 long-short 为负 | 不进入 overlay；保留为失败复用和 shadow 线索 |
| 候选池质量重排 `candidate_quality` | 4 个候选全部 rejected；短反、趋势、量价一致性和复合因子都未过门禁 | 不继续简单重排；需要更贴近事件语境的假设 |
| 状态/执行门控 `regime_execution` | `volatility_regime_compression_20d` 为 elite，10d RankIC 0.0450、ICIR 0.435 | 进入 overlay 挑战，但只作为状态/执行候选 |
| overlay 角色挑战 | `rerank_tail_risk_w010` 年化 58.72%、Calmar 8.137，高于 baseline 58.51%/8.109 | 暂列 promoted challenger，必须继续稳健性和执行复核 |
| 稳健性矩阵 | `rerank_tail_risk_w010` Calmar 胜率 85.71%，最差回撤差 0 | 标记 `robust_challenger`，下一步做执行和更长样本复验 |
| 执行复核 | `rerank_tail_risk_w010` 分钟目标价部分成交 Calmar 胜率 100%，最差回撤差为正 | 标记 `execution_robust_challenger`，进入 paper shadow 候选 |
| 冻结 CPCV | `tail_risk_soft_q10_p25` 在 2018-2026、8 块取 2 测试的 28 个组合中，测试 Calmar 胜率 96.43%，中位测试 Calmar delta 0.481824 | 可替代研究 reference baseline；正式 preset/live 仍需 paper shadow 和再次审批 |
| 梯度惩罚复验 | `tail_risk_ramp_q50_p25` 赢原始 baseline，但相对 `tail_risk_soft_q10_p25` 的测试 Calmar 胜率仅 39.29%，中位测试 Calmar delta -0.063815 | 标记 rejected mutation；不替换 `q10_p25` |
| 窄梯度复验 | `q20/p20`、`q25/p20`、`q25/p25` 均未赢 `q10_p25`；最佳 `q25/p25` 全样本略正但测试 Calmar 胜率 46.43%、中位测试 Calmar delta -0.002528 | 关闭 plain ramp 线；保留 `tail_risk_soft_q10_p25` |

关键学习：`tail_risk_downside_vol_20d` 硬过滤失败，但 10% 权重轻量重排过关。因子变形优先改变角色和接入强度，再考虑复杂公式变换。
