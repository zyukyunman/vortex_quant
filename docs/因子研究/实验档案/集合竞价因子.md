# 集合竞价因子

关联：[[量价技术因子]]、[[业绩预告漂移因子]]、[[因子评价框架]]、[[因子研究与评测全流程说明]]

## 一句话结论

集合竞价信号**不作为独立 alpha 入库**；中性化后更可靠的方向是反向执行/拥挤风险提示。目前只建议在业绩预告漂移策略 shadow 中记录命中、排队和滑点，不接入 live 默认调仓。

## 研究假设

开盘/收盘集合竞价中的异常推升、量能强度和日内竞价漂移，可能反映短线拥挤、追涨和执行不确定性。若这些状态在已有 alpha 持仓中出现，应优先被视为风险覆盖或执行可靠性诊断，而不是新的正向选股因子。

## 因子角色

| 角色 | 判断 |
|---|---|
| Alpha 排序因子 | 暂不采用。`auction_o_gap_prev_close` 试验期有正向迹象，但中性化鲁棒性未过关。 |
| 风险因子 | 保留。高开盘竞价量能强度、收盘窗口推升、收盘区间位置、竞价日内漂移的原方向更像拥挤/反向风险。 |
| 过滤/执行覆盖 | 仅 shadow。可以记录高风险桶，不应直接改变实盘目标仓。 |

## 字段映射

| 数据集 | 字段 | 用途 |
|---|---|---|
| `stk_auction_o` | `open/high/low/close/amount/volume/vwap` | 开盘集合竞价价格、量能和成交强度 |
| `stk_auction_c` | `open/high/low/close/amount/volume/vwap` | 收盘集合竞价窗口推升、位置和日内漂移 |
| `bars` | `close/amount` | T-1 收盘、20 日成交额、后验收益 |
| `valuation` | `total_mv/turnover_rate` | 市值、换手等中性化控制 |
| `forecast` | 业绩预告事件 | 用于业绩预告漂移候选池 overlay 复核 |

## 评测口径

1. 本地数据，不调用外部 API。
2. 独立因子评测区间：2025-01-02 至 2026-04-30。
3. 主过滤宇宙：非 ST、非停牌、T-1 收盘价 >= 3、T-1 20 日均成交额 >= 3000 万元、相关竞价成交额为正。
4. 中性化控制：市值、20 日成交额、换手、20 日动量、20 日波动；按日 winsorize z-score 后 OLS 残差化。
5. overlay 复核区间：2025-01-01 至 2026-04-24；基准为业绩预告漂移 100 万级 Top30、`capped_with_cash`、单票 5%、20bps，并复核开盘竞价 all-or-nothing 小资金口径。
6. `auction_o_amount_intensity` 作为同日开盘竞价风险；`auction_c_window_push`、`auction_c_range_pos`、`auction_oc_intraday_drift` 只允许 T-1 滞后使用；`auction_o_gap_prev_close` 不作为持久 alpha 或覆盖信号。

## 阶段判断

`research_lead`。集合竞价反向信号可以作为执行/风险观察项继续 shadow，但**现在不接入 live 候选默认规则**。

## 失败或保留原因

保留原因：

1. 中性化后的反向方向稳定，尤其 `auction_o_amount_intensity` 和收盘竞价推升类信号。
2. 与业绩预告漂移候选池有可解释的执行拥挤/排队风险关联。
3. 小资金 AON 口径显示跳过开盘量能强度 top10% 能压低回撤，说明有风控信息。

不晋升原因：

1. 目标权重口径 baseline 仍为最佳；过滤/半仓多数降低年化和 Calmar。
2. 小资金 AON 的 best Calmar 来自明显降仓，年化从 50.89% 降到 43.86%，不能证明增强 alpha。
3. 同日开盘集合竞价终值能否稳定用于撤改单，需要模拟盘/真实委托回报验证。
4. 收盘集合竞价信号只能 T-1 滞后使用，不能同日解释执行收益。

## 可复用经验

1. 集合竞价高量能/推升信号不要直接当追涨 alpha；在当前样本里原方向更像拥挤或执行不确定性。
2. 风险 overlay 必须同时看收益损失、回撤改善、换手和成交可得性；只提升 Calmar 但大幅牺牲年化，不足以进入 live 默认。
3. 对实盘候选，下一步应记录 shadow 命中后的委托时间、排队、成交率、成交价偏离和未成交原因，而不是先写生产规则。
