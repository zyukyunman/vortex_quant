# 峰岭谷成交量结构因子

关联：[[量价技术因子]]、[[因子研究与评测全流程说明]]

## 一句话结论

峰岭谷成交量结构在 PDF 报告中表现很强，但在 Vortex/Tushare 2024 高流动性 100 股、2026 高流动性 80 股以及 2017-2026 liquid300 十年面板中，`ridge_minutes` 正向结论都没有复现。已确认此前十年脚本的 ridge 实现与 skill/PDF 口径存在差异：脚本用“相邻高量”近似，skill/PDF 写的是“连续 3 分钟及以上放量”。但 2024 liquid100 审计显示改成连续 ≥3 分钟后，ridge 仍未恢复强正；当前更值得保留的是孤立量峰、量谷和全市场状态线索。

## 研究假设

分钟成交量在日内形成的峰、岭、谷形态可能反映资金关注、冲击吸收和趋势延续/反转结构。

## 因子角色

当前不把峰岭谷视为可裸跑的独立策略，而按三类组件继续研究：

1. 横截面排序：`peak_volume_ratio`、`valley_relative_vwap`、`neg_ridge_minutes`。
2. 过滤/二级排序：在已有 alpha 中剔除拥挤放量或优先选择量谷修复结构。
3. 择时/风险状态：全市场 `ridge/peak/valley` 聚合后做仓位缩放、半仓或因子切换。

## 字段映射

当前已用 Tushare `stk_mins` 1 分钟数据做本地复现。核心字段：

1. `symbol / date / minute / trade_time`。
2. 分钟成交量 `volume` 与成交额 `amount`。
3. 分钟价格 `open/high/low/close`。
4. 同股票、同分钟的历史成交量 rolling mean/std。
5. 未来 1d/3d/5d/10d/20d 复权收盘收益。
