# 大单净流入原始方向

关联：[[资金流因子]]、[[资金流拥挤反转因子]]

## 一句话结论

原始“大单/超大单净流入=利好”在 2023-2026 样本中方向为负，应避免重复使用该直觉假设。

## 研究假设

传统直觉认为大单/超大单净流入代表主力资金买入，未来收益更高。

## 评测口径

1. 样本：2023-01-01 至 2026-04-24。
2. 因子 shift 1 个交易日。
3. T-1 流动性过滤。
4. 评测 1d、5d、20d、60d。

## 阶段判断

`rejected`。

## 失败原因

方向与直觉相反：近三年 A 股样本中，大单/超大单净流入更像短期拥挤或追涨，而不是未来收益来源。

## 可复用经验

以后研究资金流时应先做方向检验，不要直接把“大单流入”设为正向因子。
