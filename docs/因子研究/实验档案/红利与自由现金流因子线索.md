# 红利与自由现金流因子线索

关联：[[价值质量因子]]、[[超级鹿鼎公资料集]]、[[因子研究与评测全流程说明]]

## 一句话结论

红利、自由现金流和质量类因子有稳定中长期排序价值，其中 `quality_growth_value` 的 IC 最强，但直接做 TopN 多头回撤过大；它更适合作为组合质量腿或风控过滤，不应单独宣传为低回撤策略。

## 研究假设

持续分红、股息率较高、ROE 稳定、现金流质量好的公司，长期风险收益可能更稳健，适合做组合防守腿。

## 失败或保留原因

1. `quality_growth_value` 证明质量/成长/估值复合排序有效，但 Top30/20 日调仓最大回撤约 -27%，离低回撤目标很远。
2. 股息率和 FCF 单项因子并没有形成足够强的多头收益，不能把“红利/自由现金流”简单包装成高收益策略。
3. 价值质量因子更可能降低组合极端风险、改善持仓解释性，而不是单独承担收益主引擎。

## 阶段判断

`research_lead`。PIT 财务字段已经完成一轮系统评测；方向值得保留，但只作为组合质量腿和防守过滤继续观察。

## 可复用经验

1. 这类因子不应追求短期暴利，更适合作为降低组合波动、改善回撤和提升资金容量的防守腿。
2. 单独股息率、单独自由现金流率不够强；要用质量、成长、估值、低波等多维复合。
3. IC 很强仍可能 long-only 回撤很大，必须进入组合层和市场状态层再评估。

## 下一步

1. 做持仓级 `quality_growth_value 50% + 低PB低波股息质量代理 50%` 组合回测，加入真实换手、成本和容量。
2. 做 `risk-on QGV / risk-off 半现金半防守` 仓位规则，确认 overlay 左尾改善能否在持仓级复现。
3. 与业绩预告漂移策略做持仓交集、相关性和组合层回撤测试。
4. 将低 PB、低波、质量成长组合拆成可解释子因子，避免复合因子黑箱化。
