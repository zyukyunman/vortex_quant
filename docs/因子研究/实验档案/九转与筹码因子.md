# 九转与筹码因子

关联：[[量价技术因子]]、[[因子评价框架]]、[[因子研究与评测全流程说明]]

## 一句话结论

筹码成本分布宽度比九转计数更稳定：`cost_spread_90/70` 在 20 日维度有较稳定正 IC；九转事件本身有正向事件收益差，但更像短周期状态/事件标签，不能直接当独立多头策略。

## 研究假设

1. 筹码成本分布越分散，可能代表换手充分、筹码重新定价或历史成本结构复杂，后续中短期收益存在横截面差异。
2. 低获利盘比例可能反映超跌或筹码压力释放，但也可能只是弱势股暴露。
3. 九转上/下行事件可能捕捉短周期情绪延续或反转状态，需要用事件收益、候选池 overlay 和市场状态分段验证。

## 因子角色

| 角色 | 判断 |
|---|---|
| Alpha 排序 | 保留筹码成本分布宽度，尤其 `cost_spread_90` / `cost_spread_70` 的 20 日排序线索。 |
| 事件/状态标签 | 保留九转事件，但需要作为二级状态或 shadow 标签，不直接改实盘默认规则。 |
| 风险过滤 | 低获利盘、九转计数方向需要进一步做风格正交和弱市分段，暂不接入。 |

## 关键指标

口径：2017-2026 liquid300 union，本地 `cyq_perf + stk_nineturn + bars/adj_factor`，未来 1/5/20 日复权收盘收益，按日横截面 RankIC 与 top20%-bottom20% 多空均值。

| 因子/事件 | 期限 | IC / 收益差 | 结论 |
|---|---:|---:|---|
| `cost_spread_90` | 20d | IC 0.0522、ICIR 0.503、正 IC 66.9%、LS 1.05% | 当前最稳筹码线索 |
| `cost_spread_70` | 20d | IC 0.0497、ICIR 0.519、正 IC 68.5%、LS 1.06% | 与 90% 成本宽度一致 |
| `cost_spread_90` | 5d | IC 0.0323、正 IC 61.9%、LS 0.37% | 中短期仍有排序力 |
| `winner_rate_low` | 1d | IC 0.0212，但 LS 为 -0.13% | 低获利盘有短期排序但多空不稳定 |
| `up_count_positive` | 1d | IC -0.0251 | 上涨九转计数原方向偏负，不直接追涨 |
| `nine_up_event` | 20d | 事件均值 2.99%，非事件 1.34%，差值 1.66% | 有事件状态价值，但需风格/市场分段 |
| `nine_down_event` | 20d | 事件均值 2.20%，非事件 1.36%，差值 0.84% | 也有正差值，说明九转事件可能混入市场 beta/活跃股暴露 |

## 阶段判断

`research_lead`。筹码成本分布宽度进入下一轮候选池 overlay；九转事件先作为 shadow 状态标签，不接入 live 默认交易链路。

## 失败或保留原因

保留：

1. `cost_spread_90/70` 的 20 日 IC、ICIR 和正 IC 占比在十年样本中较稳定。
2. 九转事件后 20 日收益均值高于非事件，说明它可作为事件状态或市场活跃度标签继续研究。
3. `cyq_chips` pilot 已证明可落地，但数据量较大，应等具体假设明确后再扩容。

风险：

1. 筹码成本分布可能暴露于小市值、波动、换手和活跃度，需要做风格正交。
2. 九转事件的正收益差并不能证明方向 alpha，因为上九转和下九转都为正，可能只是事件股活跃度或市场 beta。
3. 当前没有接入交易成本、容量和候选池实盘执行约束。

## 可复用经验

1. `cyq_perf` 比 `cyq_chips` 更适合作为常规日频因子源；`cyq_chips` 更适合针对明确假设做小样本结构拆解。
2. 九转不要直接解释为“上九转追涨、下九转抄底”，必须先做事件后收益、风格暴露和市场状态拆解。
3. 筹码类因子下一步应进入业绩预告漂移候选池或质量/低波组合做二级排序，而不是单独 TopN 裸跑。
