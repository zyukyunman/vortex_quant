# 业绩预告漂移因子

关联：[[事件因子]]、[[业绩预告漂移策略研究总结]]、[[业绩预告漂移策略测试报告]]、[[因子研究与评测全流程说明]]

## 一句话结论

当前最接近实盘影子跟踪的事件 alpha：业绩预告 surprise 有中等正向预测力，完整策略依赖事件窗口、市场门控和可交易过滤。

## 研究假设

A 股业绩预告发布后，市场可能不会在第一个交易日完全消化信息。预增、扭亏和利润同比大幅增长事件，在公告后 20-60 个交易日存在漂移。

## 字段映射

| 数据集 | 字段 | 用途 |
|---|---|---|
| `forecast` | `ann_date`, `p_change_min`, `p_change_max`, `type` | 事件源和 surprise 打分 |
| `bars` | `open`, `close`, `amount` | open→close 收益、流动性 |
| `stk_limit` | `up_limit`, `down_limit` | 开盘涨跌停约束 |
| `suspend_d` | `suspend_type` | 停牌过滤 |
| `stock_st` | `date`, `symbol` | ST / 风险警示剔除 |
| `index_daily` | `close` | 市场门控 |

## 评测口径

1. 公告后第 1 个可交易日开始：`delay_days=1`。
2. 持有窗口：`hold_days=40`。
3. 成本：20 bps。
4. 研究复核覆盖 2017-2026，最新实盘前置关注 2025-2026。
5. 当前实盘版本：100 万级、`stable_100w` / Top50 上限、`capped_with_cash`、单票 5%、先 shadow/paper/gated live。

| 实验 | 年化 | 最大回撤 | 结论 |
|---|---:|---:|---|
| 2025-02 至 2026-04 Top30/5% cap/20bps baseline | 66.86% | -3.94% | 当前小资金口径仍最强 |
| 最好股票级二级过滤：质量/价值过滤 | 51.74% | -4.11% | 收益和回撤都弱于 baseline |

2026-05-17 实盘缺口复审后，`tail_risk_soft_q10_p25` 不能再被视为已晋升 base。新的可信结论是：`tail_risk_downside_vol_20d` 必须 T-1 shift；同日 close/amount 不得影响同日 open→close 信号。手动 `prepare-next-session` 也必须保证 `execution_trade_date > signal_as_of`，执行组合使用执行日信号行，但特征与风控数据只来自信号日前已完成数据。实盘缺口修复不构成 alpha 改进，不能替代 CPCV。

## 失败或保留原因

2. 策略层收益强，当前是最接近实盘影子的 alpha。
3. 小资金容量相对可行。
4. 最新二级过滤实验显示 baseline 已经足够紧凑，强行叠加质量/价值/PRV 股票级过滤会削弱收益。

未晋升 production-ready 的原因：

1. 需要修正 T-1 流动性和 T-1 市场门控口径。
2. 开盘竞价容量验证只是压力测试，不是最终成交承诺。
3. 需要连续 shadow / paper 记录真实目标持仓、委托、成交和滑点。

## 可复用经验

事件 alpha 不能只看回测收益；必须同时记录公告可见时间、涨跌停、停牌、ST、容量和实盘执行延迟。

不要为了“多因子化”而强行给业绩预告漂移加股票级过滤。若要引入其他因子，优先做市场级仓位缩放、风险状态 shadow 观察，确认边际贡献后再进入实盘候选。
