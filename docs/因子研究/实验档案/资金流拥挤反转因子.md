# 资金流拥挤反转因子

关联：[[资金流因子]]、[[大单净流入原始方向]]、[[因子研究与评测全流程说明]]

## 一句话结论

大单/超大单持续净流入反向后有较好 RankIC；2024 与 PRV 分钟量状态叠加后，量谷状态可小幅改善 long-only，峰/岭高状态更像拥挤透支风险。

## 研究假设

短期大单/超大单持续净流入可能代表拥挤交易和追涨，未来 20-60 日反而有反转压力；叠加低波可以降低噪声。

| 数据集 | 字段 | 用途 |
|---|---|---|
| `moneyflow` | `buy_lg_amount`, `sell_lg_amount`, `buy_elg_amount`, `sell_elg_amount` | 大单/超大单净流 |
| `moneyflow` | `buy_sm_amount`, `sell_sm_amount` | 小单净流对照 |
| `bars` | `close`, `amount` | 收益与流动性过滤 |

## 评测口径

1. 样本：2023-01-01 至 2026-04-24。
2. 因子收盘后可见，整体 shift 1 个交易日。
3. 流动性过滤使用 T-1 的 20 日平均成交额。
4. 评测周期：1d、5d、20d、60d。
