# 小市值低波反转因子

关联：[[量价技术因子]]、[[因子研究与评测全流程说明]]

## 一句话结论

小市值 + 低波 + 短期反转的截面 IC 很强，但单独 long-only 回撤过大，不能直接作为实盘策略。

## 研究假设

小市值溢价、低波防御和短期反转可能共同提供中短期截面收益。

## 字段映射

| 数据集 | 字段 | 用途 |
|---|---|---|
| `valuation` | `total_mv` | 市值反向 |
| `bars` | `close` | 20 日波动、5 日反转、未来收益 |
| `bars` | `amount` | 流动性过滤 |

## 评测口径

1. 样本：2023-01-01 至 2026-04-24。
2. 估值和价量因子使用 T-1 可见口径。
3. 流动性过滤使用 T-1 20 日平均成交额。
4. Top30/50/80 long-only 回测，20 bps 成本。

## 阶段判断

`research_lead`。

## 失败或保留原因

保留：IC 强且稳定。
失败：多头策略最大回撤过大，风格暴露明显，不能直接作为低回撤策略。

## 可复用经验

强 IC 因子仍可能需要市场门控、行业约束、容量约束或组合层风险预算。不要把 IC 强直接等同于策略可用。
