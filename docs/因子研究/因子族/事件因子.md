# 事件因子

关联：[[因子研究档案]]、[[业绩预告漂移因子]]、[[因子研究与评测全流程说明]]

事件因子关注公告、财报、业绩预告、停复牌、龙虎榜、涨跌停等离散事件。核心要求是 PIT：必须明确事件在什么时间点可见，不能使用公告前不可见信息。

## 经验

1. 事件因子不能只看收益，必须审查公告时间、交易可得性、停牌和涨跌停。
2. 如果收益来自同日收盘后才知道的信息，直接判为 `invalid_experiment`。
3. 高收益事件策略进入实盘前，必须补 shadow 跟踪和执行回报。
