# 价值质量因子

关联：[[因子研究档案]]、[[低PB低波股息质量因子]]、[[红利与自由现金流因子线索]]、[[超级鹿鼎公资料集]]

价值质量因子关注估值安全边际、股息率、现金流、ROE 稳定性、杠杆风险和盈利质量。

## 经验

1. 价值质量因子通常不应只看 20d，应补 60d/120d。
2. 财务字段必须按公告日/生效日做 PIT。
3. 这类因子可能更适合做组合防守腿，而不是单独高年化策略。
4. 单独股息率和单独自由现金流率较弱，复合质量成长价值更值得保留。
