# 量价技术因子

关联：[[因子研究档案]]、[[小市值低波反转因子]]、[[峰岭谷成交量结构因子]]、[[动量与101量价因子]]

量价技术因子关注低波、反转、动量、成交量结构、支撑压力、分钟微观结构等。

## 经验

1. IC 强不等于可直接做多头策略。
2. 小市值、低波、反转容易有风格暴露和市场 beta，需要容量与回撤审查。
3. 分钟因子必须特别检查数据可见性、成交口径和执行延迟。
4. 101 Alpha 风格公式适合作为公式生成语法和候选池，必须在 Vortex PIT 数据中重新实现、评测和归档。
