# 资金流因子

关联：[[因子研究档案]]、[[资金流拥挤反转因子]]、[[大单净流入原始方向]]

资金流因子使用 `moneyflow`、成交额结构、大小单行为和拥挤交易代理。当前经验显示，不能简单假设“大单净流入就是利好”。

## 已归档

| 档案 | 状态 | 结论 |
|---|---|---|
| [[资金流拥挤反转因子]] | `research_lead` | 反向后 IC 较好，但 long-only 回测失败 |
| [[大单净流入原始方向]] | `rejected` | 原始方向近三年为负 |

## 经验

1. 资金流数据通常收盘后可见，应至少 shift 1 个交易日。
2. 资金流因子可能更适合作为排序、回避或对冲线索。
3. 如果 long-only 回测与 IC 相反，要先检查分组单调性、市场 beta 和行业拥挤。
