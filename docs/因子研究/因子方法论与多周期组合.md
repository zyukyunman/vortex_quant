# 因子方法论与多周期组合

关联：[[因子研究档案]]、[[因子评价框架]]、[[动量与101量价因子]]、[[业绩预告漂移因子]]

## 一句话结论

因子不是策略本身。Vortex 后续因子研究要先判断因子角色，再决定周期、组合方式、交易动作和评价标准；短线因子解决买卖点和换手，长线因子解决风格、风险预算和左尾，最终用策略组合层结果验收。

## 研究流程

| 阶段 | 目标 | 产物 |
|---|---|---|
| 公式可计算性 | 确认字段、PIT、缺失率、异常值 | 因子矩阵、覆盖率、字段映射 |
| 单因子预测力 | 判断方向、期限和半衰期 | 1/5/10/20/60 日 IC、ICIR、分层收益 |
| 暴露诊断 | 判断是不是市值/低波/流动性/行业代理 | 风格回归、残差 IC、行业分段 |
| 组合角色 | 判断排序、过滤、择时、仓位、风险或组合腿 | 角色标签和接入方式 |
| 组合接入 | 看是否改善现有 alpha | 年化、回撤、Calmar、换手、坏日收益 |
| 交易可行性 | 检查容量、成交、成本和延迟 | 成交率、冲击成本、资金规模分层 |

## 多周期结合

| 周期 | 常见因子 | 应解决的问题 | 不应承担的问题 |
|---|---|---|---|
| 长周期 60-252 日 | 质量、红利、低波、长期动量、估值 | 风格倾向、风险预算、防守腿、组合底仓 | 不负责精确买卖点 |
| 中周期 10-60 日 | 动量残差、拥挤反转、筹码成本、行业强弱 | 候选池排序、仓位倾向、持有期选择 | 不负责开盘成交细节 |
| 短周期 1-10 日 | 短反、开盘竞价、成交冲击、PRV 分钟状态 | 买点/卖点、换手控制、坏持仓剔除 | 不负责长期收益来源 |
| 日内/执行 | 集合竞价、盘口、分钟量价 | 成交可靠性、容量、滑点 | 不单独定义策略 alpha |

## 因子组合方式

1. **先分组再合成**：同类因子内部先做 rank ensemble，再与其他因子族组合，避免同一风险源重复加权。
2. **先残差再接入**：强 IC 因子若主要来自小市值/低波/低换手，应同时记录原始版和中性化残差版。
3. **条件触发**：弱市时用防守腿或半仓，强市时提高事件/动量腿权重，不把所有状态平均在一个固定公式里。
4. **短线修正长线**：长周期因子决定股票池，短周期因子只调整买入优先级、延迟买入或降低权重。
5. **策略内生状态**：如果主策略候选池质量下降、可成交率下降或坏日集中，应作为择时信号候选。

## 验收标准

单因子验收：

1. 覆盖率稳定，字段和 PIT 口径明确。
2. IC 或分层收益在目标期限有可解释方向。
3. 暴露可解释，不能误把单一风格代理包装成新 alpha。
4. 明确角色：排序、过滤、择时、仓位、风险或组合腿。

策略接入验收：

1. 至少跑基准对照、原始因子接入、残差因子接入、分市场状态接入。
2. 不能只看最大回撤下降，还要看收益牺牲、踏空成本和换手。
3. 必须纳入交易成本、调仓延迟、涨跌停/ST/停牌、容量和开盘成交。
4. 对 100 万、1000 万、5000 万、1 亿资金规模分别给判断。

## 对当前研究线的约束

1. [[动量与101量价因子]] 的强 IC 多来自低波、短反、小市值、低换手等暴露；后续必须全库复现并标注暴露，不只挑几个好看的公式。
2. [[峰岭谷成交量结构因子]] 在全 A 分钟数据完成前不下最终判断；先做 ridge 口径审计。
3. [[九转与筹码因子]] 里的筹码成本分布宽度先作为中周期候选池排序，不直接成为独立策略。
4. [[业绩预告漂移因子]] 是当前主 alpha；新增因子优先测试“是否改善它”，而不是各自裸跑争第一。

## 2026-05-02 多周期组合实证校准

Alpha101-style Batch A 的多周期组合复验支持上述方法论：

| 组合 | 更适合的角色 | 关键证据 |
|---|---|---|
| `defensive_lowrisk_combo` | 长周期防守腿 / 风险预算 | 20d IC 0.1011、60d IC 0.1218、60d 正 IC 83.30%，是最稳定的长周期腿 |
| `multi_horizon_combo` | 多周期候选组合 / overlay 待测 | 20d IC 0.0982、20d 正 IC 77.98%、20d 多空均值 1.40%，但仍需策略接入验证 |
| `medium_candidate_combo` | 中周期候选池排序 | 20d IC 0.0710、20d 正 IC 71.87%，适合候选池排序而非独立策略 |
| `anti_crowding_risk_combo` | 拥挤风险反向腿 | 20d IC 0.0699、60d IC 0.0716，可作为追涨拥挤风险过滤 |
| `short_execution_combo` | 短周期执行/换手腿 | 20d IC 0.0382、60d IC 0.0336，长期预测弱，更适合执行节奏和坏持仓剔除 |

因此，后续组合研究不应问“哪个单因子年化最高”，而应问：

1. 长周期防守腿能否降低业绩预告漂移或其他主 alpha 的左尾。
2. 中周期候选池腿能否在不大幅提高换手的情况下改善排序。
3. 短周期执行腿能否改善开仓、减仓和成交时点，而不是作为核心持有收益来源。
4. 拥挤风险腿能否在弱市或追涨拥挤阶段做仓位缩放。

## 可复用经验

1. 高 IC、低收益、多头大回撤通常不是“因子没用”，而是组合构建、市场 beta、风格暴露或交易约束没处理。
2. 低回撤策略往往来自 alpha、择时、组合约束和执行共同作用，不应期望单因子自然满足低回撤。
3. 多周期组合的关键不是把所有因子加权，而是让每个周期只处理自己擅长的问题。
