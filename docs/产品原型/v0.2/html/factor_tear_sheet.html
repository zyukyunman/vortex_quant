截面分布与稳定性

截面质量直接影响因子可靠性——分布异常或覆盖率不稳定的因子，IC 再高也不可信。需重点关注覆盖率持续 > 80%。

截面分布直方图
每期截面上因子值的分布形态。近似正态 → 好；严重右偏 → 需去极值处理
因子覆盖率时序
每期有因子值的股票数占比。持续 > 80% → 因子有实用性
因子截面自相关
相邻两期截面排名的 Spearman 相关。高（> 0.8）→ 低换手成本；低 → 变化快、换手高
截面均值 / 标准差时序
截面均值和标准差长期趋势。趋势漂移 → 因子定义随时间失效，需定期重新校准
