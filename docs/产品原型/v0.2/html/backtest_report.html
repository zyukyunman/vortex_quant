交易分析

持仓周期分布、盈亏比、最大连续亏损、换手率时序——评估策略的交易行为特征。

持仓周期分布
每笔交易的持仓天数分布。过短 → 交易成本高；过长 → 响应慢
单笔盈亏分布
每笔交易的收益率分布。右偏且均值为正 → 截断亏损、放大盈利
换手率时序
每期的双边换手率。突然飙升 → 可能是因子信号大幅变动或市场极端事件
连续盈亏统计
最大连续盈利/亏损的笔数和金额。连续亏损越长 → 心理压力越大
