# 公司运营模型

## 一句话定义

Vortex AI 量化公司操作系统把你定义为“最终负责人”，把系统定义为本地控制面，把智能体定义为岗位化团队。系统目标不是替你“聊天做研究”，而是让你像管理一家小型量化公司一样，看状态、批决策、追产物、控风险。

## 运营原则

1. **你最终负责**：资金、实盘、策略晋升、凭证、交易权限都必须由你确认。
2. **运营总协调负责推进**：系统可以主动拆任务、催进度、拉会议、给你摘要。
3. **智能体只能产出建议和结构化产物**：智能体输出必须落到智能体会议、决策、任务或产物。
4. **事实源不来自聊天**：事实源是运行注册表、产物 JSON、审计日志和版本血统。
5. **研究和实盘分权**：研究智能体可以高自主探索；交易相关智能体默认失败即阻断。

## 公司岗位

| 岗位 | 职责 | 默认状态 | 标准产出 |
|---|---|---:|---|
| 你 | 资金、方向、实盘和最终审批 | 人类 | 批准 / 驳回 / 追问 |
| 运营总协调 | 任务拆解、运行协调、状态简报、阻塞升级 | v1 激活 | 状态简报、故障分诊、会议计划 |
| 产品架构师 | 产品对象、页面、动作、用户旅程 | v1 激活 | 产品规格、验收标准 |
| UI 架构师 | 工作台布局、状态表达、视觉系统、交互密度、原型验证 | v1 激活 | 页面原型、组件规则 |
| 控制面架构师 | 运行注册表、状态机、调度、审批、审计 | v1 激活 | 契约、状态转换、接口约束 |
| 数据运营负责人 | 数据更新、质量、时点一致、快照发布 | v1 激活 | 数据健康结论、快照就绪结论 |
| 研究总监 | 研究方向、实验优先级、因子档案 | v1 激活 | 研究计划、候选队列 |
| CogAlpha 负责人 | 21 个研究视角、变异、交叉、质量门禁输入 | v1 激活 | 因子委员会结论、血统计划 |
| 策略晋升专员 | 信号、策略版本、回测、晋升路线 | v1 激活 | 策略候选备忘录 |
| 风险官 | 未来函数、过拟合、容量、回撤、实盘风险 | v1 激活 | 风险门禁结论 |
| 执行运营 | QMT、模拟盘、订单、成交、对账 | v1 激活 | 执行就绪和对账报告 |
| 归档审计员 | 产物、版本血统、复盘、历史决策 | v1 激活 | 归档清单、决策日志 |

## 权限边界

| 动作 | 智能体可自动做 | 需要你确认 |
|---|---:|---:|
| 读取本地产物和运行状态 | 是 | 否 |
| 生成研究假设、因子候选、报告摘要 | 是 | 否 |
| 运行只读诊断、离线研究脚本 | 是 | 否 |
| 新建研究 run / 因子候选 run | 是 | 可选 |
| 改实盘策略配置 | 否 | 是 |
| 提交 QMT 委托或取消真实订单 | 否 | 是 |
| 修改凭证、账户、接口密钥、交易通道 | 否 | 是 |
| 将因子晋升为 strategy candidate | 建议 | 是 |
| 将模拟盘/影子跟踪晋升实盘 | 建议 | 是 |

## 标准会议

| 会议 | 触发 | 主持 | 必须产出 |
|---|---|---|---|
| 持续运营状态简报 | 用户打开工作台、定时巡检、关键事件触发 | 运营总协调 | 总状态、阻塞、待审批、建议动作 |
| 运行故障分诊 | 任一运行失败、阻断或心跳过期 | 运营总协调 | 根因假设、证据、下一步、负责人 |
| 因子研究委员会 | 新研究方向或下一轮演化 | 研究总监 / CogAlpha 负责人 | 候选列表、反对意见、质量风险 |
| 策略晋升委员会 | 因子/信号要进入策略候选 | 策略晋升专员 | 晋升/驳回/补实验决策 |
| 交易审查 | 盘前、盘中异常、盘后对账 | 执行运营 | 交易状态、风险、是否阻断 |
| 事故复盘 | 数据、研究、交易或智能体事故 | 运营总协调 | 影响、根因、修复、预防项 |
| 每周公司复盘 | 每周固定复盘或用户要求 | 你 / 运营总协调 | 收益、风险、研究进展、下周优先级 |

## 日常运营节奏

```text
系统持续运行
  -> 调度器和智能体按项目计划推进任务
  -> 关键事件触发状态简报或飞书通知
  -> 你只处理批准、驳回、追问、暂停
  -> 运营总协调把动作转成运行和任务
  -> 各岗位智能体执行、讨论、产出结构化产物
  -> 运行中心和归档中心持续记录事实
```

## v1 激活范围

第一版全部岗位都有 skill，但系统只要求这些岗位形成闭环：

1. 运营总协调：状态简报、故障分诊和跨岗位协调。
2. 产品/UI 架构师：定义工作台页面、交互和可验证原型。
3. 控制面架构师：定义运行、任务、产物、决策契约。
4. 研究总监 + CogAlpha 负责人：组织因子研究闭环。
5. 风险官 + 执行运营：守住实盘门禁。
6. 归档审计员：保证所有结论可追溯。
