## 二、资料主题结构

PDF 目录大致分为九组内容：

1. 投资理念：价值之锚、现金与穿越牛熊、独孤九剑与易筋经、盈利模式、现金流、合理估值。
2. 技术分析与交易：搬砖、看得远但不能做得远、技术玩法弱点、做 T、完美底部、网格交易、K 线口径。
3. 大盘与个股：大盘位置、老滑头 A 股、技术与统计视角、回购、好股票、垄断、永续组合。
4. 风险：高风险与低风险相对性、风险敞口、未雨绸缪、紧急与重要。
5. 产业与周期：地产、银行金融、互联网、钢铁、医疗、粮食、工资、行业周期。
6. 煤炭与电力能源：煤炭供需、长协价、煤电关系、焦煤、煤价、火电、水电、神华与淮北。
7. 人口与趋势：人口结构、老龄化、养老、出生人口、中日韩人口下降。
8. 局势与竞争：通缩、债务、通胀、低欲望社会、越南、印度、日本股市和消费经验。
9. 闲言碎语：股市好坏、抄作业、推翻常识、认知水平与市场经验。

## 三、语录心法与量化映射

这部分是 `ludinggong` skill 的核心来源：先保留大师式判词，再把它翻译成可验证的研究问题。

| 语录 / 心法 | 投资含义 | 量化映射 |
|-------------|----------|----------|
| “价值选股，趋势选时，估值定仓，波动降本。” | 选什么、何时买、买多少、如何降成本，是四个不同决策。 | `universe_filter`、`timing_signal`、`position_sizing`、`rebalance_costdown` 分层建模。 |
| “复利的伟大，需要时间。” | 长期稳定比短期炫技更重要，少犯大错才能让复利运行。 | CAGR、最大回撤、回撤恢复期、滚动收益稳定性。 |
| “有逻辑的盈利和没逻辑的盈利，短期看类似，长期看是生死之别。” | 回测赚钱必须解释利润来源，否则可能只是噪声。 | 每个因子写经济解释、适用行业、失效场景，并做分阶段/分行业检验。 |
| “现金流才是价值之锚。” | 价值锚来自可持续现金流，而不是概念或单年利润。 | OCF、FCF、利润现金含量、资本开支、分红覆盖率。 |
| “今年分红率 10%，明年亏损呢？” | 高股息必须叠加利润底线，静态股息率不能替代经营判断。 | 股息率因子必须加 `earnings_floor_score` 和现金流压力测试。 |
| “搬砖之锚就是估值差。” | 可比资产之间的换仓赚的是估值差回归，不是追涨杀跌。 | peer group 估值分位差、基本面风险相似度、换仓阈值。 |
| “价值投资比趋势投资难 10 倍。” | 若不懂财务和行业，纯主观价投反而更危险。 | 中低估值股票池 + 趋势确认，降低对主观行业判断的依赖。 |
| “低估值，高成长是做股票的不二关键。” | 完美标的稀缺，低估稳定现金流和合理估值成长也应分桶研究。 | 价值-成长-分红三维分组，分别评测收益来源。 |
| “低估值品种在经营不发生质变的背景下，股价不涨甚至下跌，未来收益更高。” | 低估下跌可能提高隐含收益，但前提是基本面没有坏。 | `fundamental_break_flag=false` 才允许估值越低权重越高。 |
| “安全来自低估、高分红、没涨过、很难下跌的企业盈利模式。” | 安全不是低波动，而是低买价、现金回报和利润底线同时成立。 | `safety_score = value_margin + dividend_sustainability + earnings_floor + no_overheat`。 |
| “保守地活下去，远远比增长更重要。” | 杠杆行业里，风控和流动性优先于增长。 | 银行/地产/保险优先看坏账、负债期限、流动性、融资依赖。 |
| “做 T 需要时间、技术、心理素质；我可以做，不代表你可以做。” | 波动降本不是免费午餐，普通策略必须限制做 T。 | 底仓/波动仓分账，做 T 仓位上限、滑点、卖飞成本单独回测。 |
| “唯一评判标准，就是能不能继续开下去，能不能赚到钱。” | 策略没有宗教正统，只有是否适合资金、能力和周期。 | 资金容量、换手、回撤、极端年份生存能力作为硬约束。 |
| “不考虑成本的能源供应是无限的，限定成本的能源供应是有限的。” | 能源研究必须看成本曲线与政策价格约束。 | 成本分位、长协价、市场价、运力、资本开支、政策上限。 |
| “想暴富很容易，全在刑法里，澳门赌桌上。” | 不现实的收益目标会诱导策略冒险。 | 目标函数加入回撤、杠杆、尾部风险和波动惩罚，不只最大化收益。 |

## 四、沉淀为 Vortex 的策略框架

核心提炼不是四个公式，而是四道投资问诊：

| 模块 | 鹿鼎公式问题 | 对应 Vortex 研究对象 |
|------|--------------|----------------------|
| 价值选股 | 这是不是便宜的好资产？便宜是错杀还是坏了？ | 股票池、质量过滤、价值因子 |
| 趋势选时 | 趋势是在帮我，还是我在用价值理由死扛？ | 择时信号、动量因子、相对强弱 |
| 估值定仓 | 这个价格给了多少安全边际？我看错后能不能活？ | 组合权重、仓位上限、现金缓冲 |
| 波动降本 | 做 T / 搬砖 / 网格是在降本，还是在增加动作风险？ | 再平衡规则、交易约束、成本模型 |

这套框架可作为 [[实例策略开发流程]] 的扩展策略原型：在低频价值/现金流选股基础上，引入趋势择时和估值分层仓位，再通过 [[策略域设计说明书]] 的回测与 compile 流程验证。

## 五、可量化字段建议

| 字段 | 解释 |
|------|------|
| `value_margin_score` | PE、PB、股息率、FCF yield 相对历史和无风险收益的安全边际 |
| `cashflow_anchor_score` | 经营现金流、自由现金流、利润现金含量与资本开支压力 |
| `dividend_sustainability_score` | 分红覆盖率、分红稳定性、未来利润下滑风险 |
| `monopoly_score` | 行政、资源、区域、品牌、技术或成本垄断强度 |
| `earnings_floor_score` | 未来 2-5 年利润底线和恢复能力 |
| `industry_understanding_score` | 是否能解释利润来源、供需、周期、政策和竞争格局 |
| `cycle_position_score` | 行业供需、库存、产能、价格周期位置 |
| `trend_timing_score` | 均线、动量、相对强弱、趋势确认 |
| `fundamental_break_flag` | 基本面是否发生质变，是越跌越买和止错的分界 |
| `liquidity_safety_score` | 交易流动性、账户现金缓冲、融资压力 |
| `leverage_risk_score` | 企业杠杆与账户杠杆的综合风险 |
| `costdown_trade_budget` | 做 T / 搬砖 / 网格可用的波动仓预算 |

新增或组合这些因子时，应按 [[因子研究与评测全流程说明]] 做多周期 IC、多空组合、稳定性和失效场景验证。
