# 策略研究结论档案

本文件集中记录阶段性策略结论和 artifact 路径。`.github/skills/factor-mining-research/SKILL.md` 只保留研究方法、评价协议和 runner 用法，不承载长期结论。

## 2026-05-17 stable_100w 切入 100 万实盘默认

结论：用户确认后，100 万实盘默认从 `baseline_top110_large` 切到 `stable_100w`。这不是新 alpha 替代研究 reference，而是资金量适配：`baseline_top110_large` 继续作为研究 reference、大容量回滚和对照版本；`stable_100w` 作为 100 万 QMT / paper / gated live 默认版本。

实盘配置：

| 项 | 新默认 |
|---|---|
| live / handoff / prepare / auto-run preset | `stable_100w` |
| live TopN 上限 | 50 |
| 组合本金默认 | 100 万 |
| 单票上限 | 5% |
| 市值过滤 | 上一可见交易日 A 股总市值前 50% |
| 回滚版本 | `baseline_top110_large` |
| 显式 shadow/challenge | `tail_risk_soft_q10_p25` |

选择原因：`stable_100w` 在 100 万整手执行复核中 Calmar 和回撤优于 `aggressive_100w`；selection-stability 复核也显示 `stable_100w` 是 theory Calmar leader。Top50 是上限，不是强制买满；目标持仓低于 TopN 时应通过 `target_diagnostics.selection_funnel` 解释候选不足、风控阻断、权限、整手和最低金额约束。

## 2026-05-17 实盘缺口修复与 baseline 优化初筛

结论：这轮不是把新策略强行上实盘，而是先把研究/实盘口径对齐。PIT 修正后，正式研究 reference 继续使用 `baseline_top110_large`；`tail_risk_soft_q10_p25` 只保留为显式 shadow/challenge。基于当前 base 的第一轮优化尝试没有找到可直接替代的 overlay；100 万资金口径下，`stable_100w` 比 `aggressive_100w` 更适合作为小资金实盘参考，并已在用户确认后切为 100 万实盘默认。

### 实盘缺口影响

| 缺口 | 修复后语义 | 对研究回测曲线影响 | 对实盘/影子影响 |
|---|---|---|---|
| 手动 prepare 可同日信号同日执行 | `signal_as_of` 必须早于 `execution_trade_date`；盘前任务使用上一完成交易日可见数据，目标组合写成执行日 | 不改变既有 event backtest；修复的是实盘编排语义 | 影响较大。最近 80 个信号日的轻量扫描中，执行日信号行与旧信号行有 47 天不同，变化率 58.75%，合计新增 127、移除 132 |
| 已持有赢家被机械卖回 5% | 已持有且仍入选时，只对超过基础目标的股数做动量惯性保留；缺动量数据默认不保护 | 不改变研究回测；这是 live target sizing 层 | 减少“涨到 6% 就卖回 5%”的无谓卖出；退出目标池、ST、权限或质量 blocked 仍照常卖 |
| 执行日 ST 数据缺失 | 默认 fail-closed；只有显式 `allow_missing_st_data=True` 才允许缺失 | 不改变研究回测 | 防止 T 日新增 ST/退市风险被 T-1 信号日检查漏过 |
| QMT health fail 直接中断 | 健康检查失败时仍落 blocked execution artifacts | 不改变研究回测 | 便于盘前/盘中审计：`order_plan`、`pre_trade_result`、`order_intent`、`execution_report` 都可追溯 |

验证：相关策略、交易、QMT、控制台回归共 177 个测试通过；`git diff --check` 通过。

## 2026-05-06 Strategy Evaluation Protocol 复评

本段为历史结论，使用的是当时尚未统一 T-1 overlay 的口径；`tail_risk_soft_q10_p25` 的晋升判断已被 2026-05-17 PIT 修正复跑覆盖。

新评价协议加入：

- Sortino；
- CVaR 5%（平均极端亏损）；
- worst5d / worst20d；
- 最大回撤持续 / 恢复；
- 月度 / 年度胜率；
- 相对 baseline delta；
- 执行容量折损后判定。

核心结论：

| 策略 | 口径 | 年化 | 最大回撤 | Calmar | Sortino | CVaR 5% | worst20d | 判定 |

| `baseline_top110_large` | event theory | 57.76% | -8.35% | 6.9186 | 7.2615 | -1.47% | -4.19% | `reference` |
| `baseline_top110_large` | 1 亿整手 | 54.61% | -8.94% | 6.1094 | 5.9297 | -1.69% | -6.06% | `reference` |

| `liquidity_top90_1000w` | event theory | 58.09% | -8.35% | 6.9583 | 7.1862 | -1.50% | -4.23% | `compare_only` |
