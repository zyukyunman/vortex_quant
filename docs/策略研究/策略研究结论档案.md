## 2026-05-06 Strategy Evaluation Protocol 复评

新评价协议加入：

- Sortino；
- CVaR 5%（平均极端亏损）；
- worst5d / worst20d；
- 最大回撤持续 / 恢复；
- 月度 / 年度胜率；
- 相对 baseline delta；
- 执行容量折损后判定。

核心结论：

| 策略 | 口径 | 年化 | 最大回撤 | Calmar | Sortino | CVaR 5% | worst20d | 判定 |
|---|---|---:|---:|---:|---:|---:|---:|---|
| `baseline_top110_large` | event theory | 57.76% | -8.35% | 6.9186 | 7.2615 | -1.47% | -4.19% | `reference` |
| `baseline_top110_large` | 1 亿整手 | 54.61% | -8.94% | 6.1094 | 5.9297 | -1.69% | -6.06% | `reference` |
| `liquidity_top80` | event theory | 58.24% | -8.35% | 6.9764 | 7.1385 | -1.52% | -4.45% | `compare_only` |

| `liquidity_top90_1000w` | event theory | 58.09% | -8.35% | 6.9583 | 7.1862 | -1.50% | -4.23% | `compare_only` |
| `liquidity_top90_1000w` | 1000 万整手 | 53.37% | -8.94% | 5.9721 | 5.6895 | -1.74% | -7.05% | `compare_only` |

- `tail_risk_soft_q10_p25` 仍是当前最强 protocol candidate：不仅提升年化和 Calmar，也改善 Sortino 与 CVaR 5%。

- `liquidity_top80` 和 `liquidity_top90_1000w` 在本轮 protocol rerun 中是 `compare_only`，整手执行口径下 Calmar / Sortino 低于 `baseline_top110_large`。
- `top160_liquidity_only` 曾在 2025-2026 capacity-aware fusion 中表现强，但本轮未通过完整 protocol rerun，暂记为 `artifact_based_followup_required`。

## 2026-05-05 tail-risk soft trim 执行复核


## 2026-05-05 regime budget 反证

正式入口：

```bash
vortex strategy earnings-forecast regime-budget-challenge
```

结论：

- 60 日广谱指数动量弱市降仓、120 日回撤降仓和组合降仓全部降低年化且不改善最大回撤。
- 评级：`reject_regime_budgets`。

## 2026-05-05 PRV 目标池复验

正式入口：

```bash
vortex strategy earnings-forecast prv-target-pool-review
```

结论：

- all-A 2025-2026 目标池中，ridge PRV 对后续收益偏负，valley PRV 弱正向，说明 PRV 有微观结构解释力。
- 但高 ridge 持仓半权 overlay 降低年化，且未改善最大回撤或 Calmar。
- 评级：`risk_shadow_or_reject`。
- 策略动作：PRV 不进入默认 alpha；可作为拥挤风险解释、执行 warning 或下一轮 soft rerank mutation。
