# 业绩预告漂移策略测试报告

关联：[[业绩预告漂移策略研究总结]]、[[研究协作与产物治理]]、[[因子研究与评测全流程说明]]、[[策略域设计说明书]]、[[Tushare全量数据接口参考表]]

> 阅读建议：本文保留完整测试过程、参数演进和实验记录；如果只想看最终策略规则、结论、可复用能力和上线门槛，先读 [[业绩预告漂移策略研究总结]]。

## 一句话结论

`forecast_surprise + market_gate + open→close` 是当前阶段最值得拥有的 A 股日频 alpha 候选。按满仓 alpha 口径，研究主候选 `hold_days=40, top_n=80, target_exposure=1.0` 在 2017-01-04 至 2026-04-24 约 9 年回测中取得 **36.61% 年化、-10.45% 最大回撤、Sharpe 2.22、Calmar 3.50**；加入涨停买入阻断、跌停卖出阻断和 ST 剔除后，严格可交易近似口径为 **31.58% 年化、-10.61% 最大回撤、Sharpe 1.97、Calmar 2.98**。

进一步复盘“为什么经常空仓”后，发现空仓几乎全部来自市场 risk-off 门控，而不是没有业绩预告信号。将市场门控从 `20/60` 调整为更快的 `10/40` 后，v2 严格可交易口径升级为 **42.32% 年化、-10.61% 最大回撤、Sharpe 2.56、Calmar 3.99**，评级为 **S**。

继续对市场门控做参数冻结 walk-forward 后，训练段两次都选中更快的 `5/20` 门控；测试段 2020-2022 与 2023-2026 均保持 S 级。v3 严格近似可交易口径升级为 **58.24% 年化、-10.79% 最大回撤、Sharpe 3.35、Calmar 5.40**。

在全历史补齐 `stk_limit` / `suspend_d` 后，v3 精确可交易复核进一步得到 **67.63% 年化、-10.69% 最大回撤、Sharpe 3.69、Calmar 6.32**；单票 3% 上限安全版为 **62.59% 年化、-8.56% 最大回撤、Sharpe 3.69、Calmar 7.31**。这说明收益不是靠“涨停买不到却强行买入”伪造出来的，但默认版在 1 亿元本金下 P99 参与率约 8.43%，仍需要影子跟踪和冲击成本复核。

该策略不再按“单策略最大回撤必须小于 5%”一票否决，因为它适合作为 alpha 袖珍策略或子策略，实际组合风险可由多品种、现金、对冲或组合层风险预算控制。

补充因子评测后，forecast surprise 本身是正向但不算极强的事件因子：1/5/20/40/60 日 IC 均值分别约 0.0290、0.0234、0.0135、0.0277、0.0530，60 日正 IC 占比约 64.79%。因此当前更准确的定性是“中等强度事件因子 + 简单规则模型 + 市场门控”，不是纯粹强因子弱模型。

## 策略灵感与挖掘过程

### 1. 为什么从业绩预告切入

前期路径验证显示，常见低频因子和技术择时难以同时获得高年化与低回撤：

| 路径             | 结论                      |
| -------------- | ----------------------- |
| 红利/价值/低波/质量    | 收益可解释，但 alpha 不够强       |
| 季度股票池 + 周频技术择时 | 可以改善交易节奏，但无法躲开系统性下跌     |
| 指数市场门控         | 能减少部分熊市暴露，但收益也被明显削弱     |
| 行业轮动           | 当前静态申万一级口径下没有形成有效 alpha |
| 龙虎榜/涨停同日信号     | 数字漂亮但存在未来函数或盘中数据依赖      |

这说明继续穷举旧参数没有意义，需要换成信息含量更强、出现频率更低、能形成短中期漂移的新信息源。

业绩预告 `forecast` 具备三个优势：

1. **信息增量强**：预增、扭亏、利润同比高增长是明确基本面事件。
2. **时点清晰**：`ann_date` 可用于事件时点管理，适合做 PIT 审查。
3. **A 股特征明显**：公告后信息扩散不一定一天完成，存在公告后漂移假设。

### 2. 策略假设

```text
业绩预告公告
  → 公告后下一个交易日才允许使用
  → 按利润增长区间与公告类型构建 surprise 分数
  → 只在核心指数风险状态 risk_on 时交易
  → 选取事件池中分数靠前股票
  → 用 open→close 收益验证次日可执行日频策略
```

这不是原始“50 只股票池买 30 只”的季度策略，而是独立的日频事件驱动策略。它赚的是公告后信息漂移，不是长期市场 beta。

## 策略逻辑详解

这套策略每天按下面顺序运行：

1. **读公告事件**：从 Tushare `forecast` 读取业绩预告。只用 `ann_date` 已经披露后的信息，不用未来财报数据。
2. **确定可交易日**：公告日可能是周末或非交易日，所以先映射到公告后的第 1 个可交易日；这一天才允许进入信号。
3. **计算 surprise 分数**：用 `p_change_min` 和 `p_change_max` 的平均值代表业绩增长幅度，再叠加公告类型权重，例如“预增”“扭亏”为正，“预减”“首亏”为负。
4. **流动性与 ST 过滤**：用 20 日平均成交额过滤低流动性股票，并剔除 ST / 风险警示股票。
5. **信号延续**：公告后漂移不是只持续一天，信号默认延续 `hold_days=40` 个交易日；在窗口内每天仍参与排序。
6. **市场门控**：用沪深300、中证500、中证1000的动量和支撑位判断 `risk_on`。只有市场 risk-on 时才允许建仓；risk-off 时目标仓位清零。
7. **截面选股**：每天在仍有效的事件池里选 surprise 排名前 `top_n=80` 的股票，等权配置，单票约 1.25%。
8. **可交易约束**：当天涨停的股票不允许新增/加仓；当天跌停的股票不允许减仓/卖出；这避免把买不到或卖不掉的交易当成收益。
9. **收益计量**：用复权 `open→close` 收益计入净值，并扣除 20 bps 交易成本。

所以它不是“全年一直满仓的股票池策略”，而是：

```text
公告事件池存在
  AND 市场 risk-on
  AND 通过流动性/ST/涨跌停约束
  => 才持仓
否则空仓或部分仓位
```

## v2 市场门控升级

用户观察到策略经常空仓后，对空仓来源做了专项诊断。严格可交易 v1 的 2258 个回测日中，空仓 1136 天，几乎全部由市场 `risk_off` 触发；`risk_on` 但无合格业绩预告信号导致的空仓为 0 天。这说明空仓主要来自市场门控，而不是事件池断档。

因此测试了多组市场门控：

## v3 参数冻结市场门控

为了避免把 `10/40` 当成事后调出来的漂亮参数，又对市场门控做了单独 walk-forward。候选网格只围绕风险开关节奏变化，不重新优化事件信号本身：`5/20`、`10/40`、`15/45`、`20/60`、`30/90`、`40/120`，以及少量确认指数数、动量阈值和支撑破位容忍度变体。

结果显示，两段训练都选中了 `momentum_window=5, support_window=20, confirmations=2`，而不是上一版默认的 `10/40`：

|---|---|---|---|---:|---:|---:|---:|---|
| WF-1 | 2017-2019 | 5/20，2 指数确认 | 2020-2022 | 52.56% | -5.80% | 3.29 | 9.06 | S |

这个结果的含义不是“越快越好”，而是业绩预告漂移本身是中短期事件 alpha，市场风控如果太慢，会错过公告后最有价值的恢复窗口。`5/20` 更像“快速恢复 risk-on”，而不是简单增加仓位；v3 平均仓位为 51.96%，相比 v2 的 50.44% 只小幅提高，但收益效率显著改善。

3. 7 月市场大盘修复但中证1000为负，策略小幅亏损，说明事件池暴露更接近中小盘和主题成长，不完全跟随沪深300。

5. 后续优化不应针对 2023 单年硬调参数，而应围绕食品/小金属/环保等负贡献行业做质量、估值、资金流过滤，并用 walk-forward 验证。

## 质量、估值、资金流二次过滤

基于 2023 负贡献复盘，测试了几类简单、可解释、PIT 近似安全的二次过滤：

1. 估值：`pb < 8`、`0 < pe_ttm < 100`。
2. 规模/容量：总市值截面排名前 60%。
3. 资金流：过去 5 日 / 20 日主力净流入为正。
4. 财务质量：`roe > 0`、扣非 ROE 为正、毛利率为正、经营现金流/收入为正、资产负债率低于 70%。

代表结果：

| 过滤器 | 年化 | 最大回撤 | Sharpe | Calmar | 2023 收益 | P99 参与率 | 结论 |
|---|---:|---:|---:|---:|---:|---:|---|
| v3 默认 | 58.24% | -10.79% | 3.35 | 5.40 | 16.01% | 8.60% | 默认研究候选 |
| `pb < 8` | 59.53% | -10.88% | 3.31 | 5.47 | 19.19% | 8.97% | 小幅改善收益，但容量尾部变差 |
| `debt_to_assets < 70` | 59.25% | -10.88% | 3.35 | 5.45 | 18.29% | 9.02% | 小幅改善收益，但不是决定性 |
| 总市值前 60% | 58.02% | -11.82% | 3.36 | 4.91 | 17.39% | 7.02% | 容量改善，回撤效率下降 |
| 5 日资金流为正 | 48.53% | -11.15% | 2.81 | 4.35 | 8.93% | 10.23% | 明显变差 |
| 20 日资金流为正 | 50.71% | -12.12% | 2.84 | 4.18 | 11.09% | 10.43% | 明显变差 |

结论：

1. 不把资金流过滤纳入默认；它减少了有效事件，收益和 2023 表现都变差。
2. 不把复杂财务质量组合纳入默认；它没有改善回撤，且容易把事件 alpha 变成低频财务因子混合体。
3. `pb < 8`、`debt_to_assets < 70` 可以作为后续候选特征，但当前增量不够大，不单独升级。
4. `min_avg_amount=500000` 是目前最值得继续观察的容量友好版本：它不是质量过滤，但显著降低参与率尾部，并改善全样本回撤效率。由于流动性阈值 walk-forward 第二段仍选中 30000，暂不替换默认，只作为 shadow trading 并行配置。

## 涨跌停与 ST 可交易性复核

用户提出的核心风险是：策略高收益是否来自“涨停买入”这种实际买不到的成交。为此新增保守执行约束：

1. `limit_list_d.limit == "U"`：当天不允许新增或加仓该股票。
2. `limit_list_d.limit == "D"`：当天不允许减仓或卖出该股票。
3. `stock_st`：ST / 风险警示股票从候选池剔除。
4. `stk_limit` 和 `suspend_d` 已加入数据注册和策略精确约束入口；`stk_limit` 必须使用未复权开盘价比较涨跌停价，避免与复权价格混用。当前已完成 2017-2026 全历史补抓，并新增精确可交易复核报告。

当前“做对了”的地方：

1. 使用公告后第 1 个交易日，避免公告前收益泄漏。
2. 用 `open→close` 而不是同日收盘信号，执行口径相对保守。
3. 用市场门控过滤系统性风险暴露，并通过 walk-forward 将默认门控从 v2 的 `10/40` 升级到 v3 的 `5/20`。
4. 用 `top_n=80` 分散事件噪声，避免单票主导。
5. 加入涨跌停/ST 约束后仍保持 A 级，说明 alpha 不完全依赖不可成交假设。

当前“做错或不足”的地方：

1. 全历史 `stk_limit` / `suspend_d` 已完成真实精确约束回测，但仍需 shadow trading 验证当期数据延迟和真实开盘成交。
2. 默认流动性阈值下尾部参与率仍偏高，v3 在 1 亿元本金下 P99 约 8.43%。
3. surprise 分数只看增长区间和公告类型，尚未结合盈利质量、估值、行业预期和资金流。
4. 2023 年相对弱但已完成专项复盘；质量/估值/资金流过滤已初测，暂不升级为默认。
5. 北交所和科创板贡献较高，需要后续确认实盘权限、滑点和容量。

上线节奏建议：

1. **现在开始 shadow trading / paper trading**：每天按真实数据生成目标持仓，但不下实盘单，用于检查数据延迟、公告更新、涨跌停和持仓稳定性。
2. **不要在 2026 年 5 月直接满额上线**：当前日期接近年报/一季报披露尾声，事件密度会下降，不适合用低事件密度阶段验证收益能力。
3. **更合理的小规模上线窗口是 2026 年半年度业绩预告前后**：在 6 月底前跑完影子跟踪和当期公告演练，若仍保持 A 级，再以小资金、低参与率上线。
4. **上线前硬门槛**：全历史精确约束已通过；下一步要求 1 亿元模拟本金下 P99 参与率最好压到 5% 附近、30 bps 成本压力不崩、当期 shadow trading 没有数据延迟或不可成交异常。

## 无效路径复盘

| 方法 | 失败原因 |
|---|---|
| 继续调季度池 + 技术择时 | 收益/回撤无法同时满足，系统性风险难靠个股支撑压力消除 |
| 常规低频价值/质量/红利 | 解释性强但 alpha 不够 |
| 行业轮动 | 当前行业动量口径没有形成有效收益 |
| 龙虎榜当日收盘买 | 结果可能很漂亮，但通常收盘后披露，未来函数风险高 |
| 涨停强度同日收盘买 | 需要分钟/实时事件流支撑，当前权限下不能作为最终结论 |
| 热榜共振 | 没有显著改善业绩预告策略的回撤效率 |
| 资金流正向过滤 | 明显降低事件覆盖，年化、Calmar 和 2023 表现均变差 |
| 复杂财务质量组合 | 未改善回撤效率，容易引入低频财务因子混合和过拟合 |
| 月份硬开关 | 可降低近期回撤但牺牲年化，且训练/测试稳定性不足 |
| 强行把回撤压到 5% | 会牺牲策略 alpha；更适合在组合层控风险 |

这些路径后续不应重复穷举同类参数，除非新增数据、执行口径或核心假设。
