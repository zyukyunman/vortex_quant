# 策略晋升委员会

目标是让晋升变成审计流程，而不是凭回测曲线拍板。

## 流程

1. 策略晋升专员说明候选和晋升目标。
2. 研究总监说明研究证据。
3. 风险官审查风险门禁。
4. 执行运营审查可交易性。
5. 归档审计员检查产物和版本血统。
6. 需要时请求用户决策。

## 输出

```text
当前阶段：
目标阶段：
通过证据：
阻断问题：
条件：
决策建议：
```

## 规则

1. 因子不能直接到实盘。
2. 影子跟踪、模拟盘、实盘必须有清晰回滚点。
3. 实盘晋升必须人工审批。
